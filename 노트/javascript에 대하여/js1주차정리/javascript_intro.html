    자바스크립트 개요
    
                    
    
                    
    자바스크립트
    

                    자바스크립트(JavaScript)는 객체(object) 기반의 스크립트 언어입니다.
                    
                    HTML : 웹 내용 작성
                    CSS : 웹 디자인
                    JavaScript : 웹 동작 구동
                    
                    자바스크립트는 웹 브라우저에서 사용되나, Node.js와 같은 프레임워크를 사용하면 서버 측에서도 사용 할 수 있다.
                    현재 대부분 컴퓨터, 스마트폰 등에 포함된 웹 브라우저에는 자바스크립트가 내장 되어 있다.
